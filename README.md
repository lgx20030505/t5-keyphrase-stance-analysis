# T5 Keyphrase-Based Political Stance Classification

## Overview

This project evaluates whether FLAN-T5 can identify political stance from generated summaries through a two-stage pipeline:

1. Extract stance-related keyphrases
2. Predict stance from those keyphrases

## Dataset

Political summaries generated by multiple LLMs.

Classes:

- Left
- Right
- Neutral

## Model

google/flan-t5-base

## Pipeline

Summary
   ↓
T5 Keyphrase Extraction
   ↓
Keyphrases
   ↓
T5 Stance Classification
   ↓
Evaluation

## Results

Accuracy: 36.67%

The model showed strong bias toward left predictions and struggled to identify right-leaning summaries.
